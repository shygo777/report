🔀 合併排序法

Merge Sort - 深入理解與可視化

主頁 氣泡排序 堆積排序

📚 原理說明

合併排序法（Merge Sort）是一種分治演算法，將輸入分成兩個半部分，對它們遞迴排序，然後合併已排序的兩個半部分。


核心概念：

  • 分治策略：將數組不斷分割成更小的部分
  • 遞迴排序：對每個子數組進行遞迴排序
  • 合併過程：將兩個已排序的數組合併成一個有序數組
  • 時間複雜度始終為 O(n log n)
  • 穩定排序：相等元素的相對位置不變

⚙️ 排序過程

演算法步驟：

  1. 分割：將陣列從中點分成兩個子陣列
  2. 遞迴排序：對左半部分遞迴調用合併排序
  3. 遞迴排序：對右半部分遞迴調用合併排序
  4. 基底情況：當陣列大小為 1 時，它已經是有序的
  5. 合併：將兩個已排序的子陣列合併成一個大的有序陣列

圖示示例：

初始陣列： [38, 27, 43, 3, 9, 82, 10]

分割過程：

[38, 27, 43, 3, 9, 82, 10]
            ↓
[38, 27, 43]    [3, 9, 82, 10]
    ↓                      ↓
[38] [27, 43]    [3, 9] [82, 10]

合併過程：

步驟 操作 結果
1 合併 [27] 和 [43] [27, 43]
2 合併 [38] 和 [27, 43] [27, 38, 43]
3 合併 [3, 9] 和 [10, 82] [3, 9, 10, 82]
4 合併 [27, 38, 43] 和 [3, 9, 10, 82] [3, 9, 10, 27, 38, 43, 82]

💻 代碼實現

完整版本：

function mergeSort(arr) { // 基底情況：陣列大小為 1 或 0 if (arr.length <= 1) return arr; // 分割陣列 const mid = Math.floor(arr.length / 2); const left = arr.slice(0, mid); const right = arr.slice(mid); // 遞迴排序並合併 return merge(mergeSort(left), mergeSort(right)); } function merge(left, right) { const result = []; let i = 0, j = 0; // 比較並合併兩個已排序的陣列 while (i < left.length && j < right.length) { if (left[i] <= right[j]) { result.push(left[i++]); } else { result.push(right[j++]); } } // 添加剩餘元素 result.push(...left.slice(i)); result.push(...right.slice(j)); return result; }

使用示例：

// 使用示例 let numbers = [64, 34, 25, 12, 22, 11, 90]; console.log("原始陣列:", numbers); let sorted = mergeSort([...numbers]); console.log("排序後:", sorted); // 輸出: [11, 12, 22, 25, 34, 64, 90]

🔍 應用場景與對比

何時使用合併排序：

  • ✓ 需要穩定排序 - 相等元素順序保持
  • ✓ 大規模數據 - 時間複雜度有保障
  • ✓ 外部排序 - 數據無法全部加載到內存
  • ✓ 並行處理 - 易於並行化
