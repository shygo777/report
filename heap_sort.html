⬆️ 堆積排序法

Heap Sort - 深入理解與可視化

主頁 氣泡排序 選擇排序

📚 原理說明

堆積排序法（Heap Sort）是一種利用堆這一數據結構所設計的一種排序算法。堆積排序的時間複雜度為 O(n log n)，是一種高效的排序算法。


核心概念：

  • 堆是一個完全二叉樹結構
  • 最大堆：每個父節點都大於或等於其子節點
  • 最小堆：每個父節點都小於或等於其子節點
  • 堆排序使用最大堆將數組排序為升序
  • 通過反複提取堆頂（最大元素）來達成排序

⚙️ 排序過程

演算法步驟：

  1. 建立堆：將輸入數組轉換成最大堆
  2. 交換根節點：將堆頂（最大值）與堆的最後一個元素交換
  3. 縮小堆：堆的大小減 1，不再考慮已排序的最後一個元素
  4. 維護堆性質：對新的堆頂進行 heapify 操作恢復最大堆性質
  5. 重複過程：重複步驟 2-4，直至堆的大小為 1

圖示示例：

初始陣列： [4, 10, 3, 5, 1]

第一步 - 建立最大堆：

        10
        /   \
        5    3
        /  \
        4    1

第二步及以後 - 提取最大值並調整：

步驟 操作 陣列狀態 已排序部分
1 交換 10 和 1，調整堆 [5, 4, 3, 1, 10] [10]
2 交換 5 和 1，調整堆 [4, 1, 3, 5, 10] [5, 10]
3 交換 4 和 3，調整堆 [3, 1, 4, 5, 10] [4, 5, 10]
4 交換 3 和 1 [1, 3, 4, 5, 10] [3, 4, 5, 10]

💻 代碼實現

完整版本：

function heapSort(arr) { const n = arr.length; // 建立最大堆 for (let i = Math.floor(n / 2) - 1; i >= 0; i--) { heapify(arr, n, i); } // 提取堆中的元素 for (let i = n - 1; i > 0; i--) { // 交換堆頂和最後一個元素 let temp = arr[0]; arr[0] = arr[i]; arr[i] = temp; // 對縮小的堆進行 heapify heapify(arr, i, 0); } return arr; } function heapify(arr, n, i) { let largest = i; const left = 2 * i + 1; const right = 2 * i + 2; // 比較左子孫 if (left < n && arr[left] > arr[largest]) { largest = left; } // 比較右子孫 if (right < n && arr[right] > arr[largest]) { largest = right; } // 如果最大值不是根節點 if (largest !== i) { let temp = arr[i]; arr[i] = arr[largest]; arr[largest] = temp; // 遞迴調用 heapify(arr, n, largest); } }

使用示例：

// 使用示例 let numbers = [64, 34, 25, 12, 22, 11, 90]; console.log("原始陣列:", numbers); let sorted = heapSort([...numbers]); console.log("排序後:", sorted); // 輸出: [11, 12, 22, 25, 34, 64, 90]

🔍 應用場景與對比

何時使用堆積排序：

  • ✓ 大規模數據排序 - 時間複雜度穩定
  • ✓ 需要保證最壞情況時間複雜度 - 始終 O(n log n)
  • ✓ 內存受限的系統 - 原地排序
  • ✓ 實現優先隊列數據結構

與其他排序算法對比：

算法 平均時間複雜度 最壞情況 空間複雜度 穩定性
