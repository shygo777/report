🔄 氣泡排序法

Bubble Sort - 深入理解與可視化

📚 原理說明

氣泡排序法（Bubble Sort）是一種簡單的排序算法，它通過重複地遍歷列表，比較相鄰元素，並根據排序順序交換它們來工作。


算法名稱來源：在排序過程中，較大的元素像氣泡一樣逐漸「浮起」到列表的末尾，最小的元素則沉向列表的開始。


核心概念：

  • 比較相鄰的兩個元素
  • 如果順序錯誤則交換它們
  • 重複此過程直到列表有序
  • 每次完整遍歷後，最大元素會移動到最後

⚙️ 排序過程

演算法步驟：

  1. 初始化：設置外層迴圈，遍歷數組 n 次（n 是數組長度）
  2. 內層迴圈：在每次外層迴圈中，比較相鄰的元素對
  3. 比較判斷：如果左邊元素 > 右邊元素，則交換它們
  4. 優化：每次外層迴圈後，最大元素已正確位置，可減少比較次數
  5. 完成：當沒有發生交換或完成所有迴圈時，排序完成

圖示示例：

初始陣列： [5, 2, 8, 1, 9]

第一次遍歷：

步驟 比較 動作 陣列狀態
1 5 vs 2 交換 [2, 5, 8, 1, 9]
2 5 vs 8 不交換 [2, 5, 8, 1, 9]
3 8 vs 1 交換 [2, 5, 1, 8, 9]
4 8 vs 9 不交換 [2, 5, 1, 8, 9]

結果：最大的 9 已到達正確位置（末尾）

💻 代碼實現

基礎版本：

function bubbleSort(arr) { const n = arr.length; // 外層迴圈控制次數 for (let i = 0; i < n - 1; i++) { // 內層迴圈進行比較和交換 for (let j = 0; j < n - 1 - i; j++) { if (arr[j] > arr[j + 1]) { // 交換相鄰元素 let temp = arr[j]; arr[j] = arr[j + 1]; arr[j + 1] = temp; } } } return arr; }

優化版本（提前終止）：

function bubbleSortOptimized(arr) { const n = arr.length; for (let i = 0; i < n - 1; i++) { let swapped = false; // 檢查是否發生交換 for (let j = 0; j < n - 1 - i; j++) { if (arr[j] > arr[j + 1]) { let temp = arr[j]; arr[j] = arr[j + 1]; arr[j + 1] = temp; swapped = true; // 標記發生了交換 } } // 如果沒有交換，說明已排序，提前終止 if (!swapped) break; } return arr; }

使用示例：

// 使用示例 let numbers = [64, 34, 25, 12, 22, 11, 90]; console.log("原始陣列:", numbers); let sorted = bubbleSortOptimized([...numbers]); console.log("排序後:", sorted); // 輸出: [11, 12, 22, 25, 34, 64, 90]

🔍 應用場景與對比

何時使用氣泡排序：

  • ✓ 教育用途 - 學習排序算法基礎
  • ✓ 小規模數據 - 數據量在 10-100 個元素左右
  • ✓ 幾乎有序的數據 - 使用優化版本時效率較好
  • ✓ 需要穩定排序且要求原地排序的場景
